# JStamp via FastAPI

To test FastAPI with a basic implementation of this module:

```sh
git clone https://github.com/WebReflection/jstamp.git
cd jstamp/test/FastAPI
. run
```

Reach the page and check the console or the network panel.

Feel free to change content in the [index.html script](./public/index.html) or to enrich the [exposed python module](./module.py) capabilities.
